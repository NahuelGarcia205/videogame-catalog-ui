# 🎮 GameStore - E-commerce Storefront

Este proyecto es la maquetación responsiva de una interfaz de tienda online de videojuegos. El objetivo principal de este desarrollo fue dominar el uso de layouts complejos con CSS (Flexbox), semántica estructurada en HTML5 y diseño adaptable a cualquier tipo de pantalla (Mobile First / Responsive Design).

 Podés ver el diseño en vivo acá: https://nahuelgarcia205.github.io/videogame-catalog-ui/

---

## 🚀 Características del Proyecto

* **Diseño Gamer Premium:** Estética oscura utilizando gradientes radiales y lineales con contrastes en rojo y púrpura.
* **Header Pegajoso (Sticky Header):** El menú principal y el buscador te acompañan flotando a lo largo de toda la navegación sin romper el flujo visual.
* **Navegación Interna Fluida (Smooth Scroll):** Uso de anclas HTML conectadas a secciones semánticas para un desplazamiento suave entre categorías.
* **Buscador Integrado:** Barra de búsqueda compacta ubicada estratégicamente en el menú superior con efectos dinámicos de enfoque (`focus transition`).
* **Catálogo en Grilla Elástica:** Tarjetas de juegos alineadas milimétricamente que se reorganizan solas mediante propiedades avanzadas de Flexbox.
* **Totalmente Responsivo:** Adaptación completa para celulares y tablets mediante Media Queries, reorganizando el menú verticalmente para pantallas chicas.

---

## 🛠️ Tecnologías Utilizadas

* **HTML5:** Estructuración semántica avanzada (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
* **CSS3:** * Sistema de cajas elásticas (**Flexbox**).
  * Posicionamiento avanzado (`position: sticky`, `z-index`).
  * Transiciones y efectos hover dinámicos.
  * Diseño adaptable (**Media Queries**).
